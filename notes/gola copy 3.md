# Apuntes de JavaScript

## Variables
Las variables se declaran con `let`, `const` o `var`.
```js
let nombre = "Juan";
const edad = 25;
```

## Funciones
Las funciones son bloques de código reutilizables.
```js
function saludar(nombre) {
    return `Hola, ${nombre}!`;
}
```
