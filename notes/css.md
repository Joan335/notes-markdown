# Apuntes de CSS

## Selectores
Los selectores apuntan a los elementos HTML.
```css
.clase { color: red; }
#id { font-size: 16px; }
```

## Flexbox
Flexbox permite alinear elementos fácilmente.
```css
.contenedor {
    display: flex;
    justify-content: center;
    align-items: center;
}
```
